# Security Policy

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report via a private GitHub Security Advisory in this repository.
Include:

- affected rule(s) and version
- impact and possible exploit scenario
- reproduction steps
- any mitigation already attempted

We will acknowledge reports and work on a fix as quickly as possible.

## Supported Versions

Security fixes are generally applied to the latest published release.
